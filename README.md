# BrowserAPI
What this is: 

This a custom homepage that can display the time of where ever you are sitting, display the weather of where ever you are sitting, and sets the background as NASA's APOD (Astronomy Picture of the Day) alongside the explanation text.

How I built it:

I built this project using the guide found on Macondo docs, learning and following every step. Once I learnt how to use APIs, I added in my own twist: making it display the picture explanation as well, which was pulled from the exact same API the guide used.

To try it yourself:

Simply go to aquaritic.github.io/BrowserAPI and if it's white, just wait a bit and refresh.
